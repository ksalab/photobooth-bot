# Photobooth Income Calculator Bot 📸 💰

Telegram-бот для автоматичного розрахунку потенційного доходу від встановлення фотокабінки в закладах (ресторани, бари, готелі).

## ✨ Функціонал

- **Покрокове опитування**: Збір даних про місто, тип закладу та кількість гостей.
- **Розрахунок сценаріїв**: Автоматичне генерування консервативного (7%), реалістичного (10%) та потенційного (15%) прогнозів доходу.
- **Генерація лідів**: Збереження всіх відповідей користувачів у стилізований Excel-файл.
- **Адмін-панель**: Миттєве сповіщення адміністраторів про новий розрахунок та можливість завантажити актуальну базу одним кліком.
- **Маркетинговий блок**: Надсилання PDF-каталогу та кнопка зв'язку з менеджером.

## 🚀 Встановлення

1. **Клонуйте репозиторій**:

    ```bash
    git clone https://github.com/ksalab/photobooth-bot.git
    cd photobooth-bot

2. **Встановіть залежності**:

    ```bash
    pip install aiogram pandas python-dotenv openpyxl
    ```

3. **Налаштуйте `.env`**:

    Створіть файл .env та додайте ваші дані:

    `BOT_TOKEN`, `ADMIN_IDS` (через кому), `MANAGER_URL`, `DB_PATH`, `FILE_PATH`.

4. **Запустіть бота**:

    ```bash
    python main.py
    ```

## 🛠 Технології

- Python 3.10+
- Aiogram 3.x (FSM, Handlers)
- Pandas & Openpyxl (Excel-звіти)
